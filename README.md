# ChatApp
The idea is to create a real time chat app using the tech we have learnt.

**Stack:** 
- CI/CD pipeline: GitHub Actions
- CI: node.js
- Security: njsscan
- Deploy: Localhost

- Container: Docker 
- Backend: Node.js, SQL(PostgreSQL)
- Middle: Apache Kafka, Express, Postman
- Frontend: JS, React
